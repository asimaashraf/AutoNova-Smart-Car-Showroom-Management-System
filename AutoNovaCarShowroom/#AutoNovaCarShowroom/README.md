# 🚗 AutoNova - Smart Car Showroom Management System

AutoNova is a desktop-based **Smart Car Showroom Management System** developed using **C# Windows Forms, PostgreSQL, Python FastAPI, Machine Learning, and Groq AI**.

The system automates showroom operations through role-based authentication, inventory management, customer bookings, AI-powered car recommendations, and machine learning-based car price prediction.

---

## ✨ Key Features

- 🔐 Role-Based Authentication (Admin, Employee & Customer)
- 🚗 Car Inventory Management
- 📋 Customer Booking System
- 📖 Booking History
- 👨‍💼 Admin Dashboard
- 👨‍🔧 Employee Dashboard
- 🤖 AI Car Advisor (Groq API)
- 💰 Machine Learning Price Prediction
- 🗄 PostgreSQL Database Integration
- 🔄 Complete CRUD Operations

## ⚙ Installation

### 1. Clone Repository

```bash
git clone https://github.com/asimaashraf/AutoNova-Car-Showroom-Management-System.git
```

### 2. Install Required Software

- Visual Studio
- PostgreSQL
- pgAdmin
- Python 3.x

### 3. Install Required Python Packages

```bash
pip install fastapi
pip install uvicorn
pip install pandas
pip install numpy
pip install scikit-learn
```

### 4. Create Database

```sql
CREATE DATABASE csms;
```

Run the provided **database.sql** script.

### 5. Start Backend

```bash
python api.py
```

### 6. Run Frontend

Open the project in **Visual Studio** and click **Start**.

## 🤖 AI & Machine Learning

### AI Car Advisor

Provides intelligent car recommendations based on:

- Budget
- Preferred Brand
- Car Type

using **Groq AI (LLaMA Model)**.

### Car Price Prediction

Predicts estimated car prices based on:

- Brand
- Model
- Year
- Mileage
- Engine Type
- Fuel Type
- Transmission

using a trained **scikit-learn** machine learning model.
